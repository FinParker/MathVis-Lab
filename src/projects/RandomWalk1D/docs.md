# 一维随机游走 (1D Random Walk)

### 什么是一维随机游走？

一维随机游走是最简单的随机过程之一。想象一个粒子在直线上运动，初始位置在原点 $x=0$。
在每一个时间步长，它以 $1/2$ 的概率向右移动一步 (+1)，或以 $1/2$ 的概率向左移动一步 (-1)。

### 关键特性

1.  **期望值为 0**：由于对称性，$\mathbb{E}[X_n] = 0$。即平均而言，粒子留在原点。
2.  **均方位移 (MSD)**：虽然平均位置是 0，但粒子会随着时间推移扩散开来。
    $$ \text{MSD}(n) = \mathbb{E}[X_n^2] = n $$
    这意味着扩散的**标准差**与步数的平方根成正比：$\sigma \propto \sqrt{n}$。

### 可视化说明

*   **轨迹图**：展示了多条样本路径随时间的变化。横轴是时间 $t$ (步数)，纵轴是位置 $x$。
*   **均方位移图**：展示了实际模拟得到的 $X_n^2$ 的平均值随时间的变化，并与理论直线 $y=x$ 进行对比。
